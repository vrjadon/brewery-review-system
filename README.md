
# Brewery Review System

This project is a Brewery Review System built using React and Firebase. It utilizes the Open Brewery DB API to fetch brewery information and allows users to review and rate breweries.

## Features

-   **Search Breweries**: Users can search for breweries by city, name, or type using the Open Brewery DB API.
-   **View Brewery Details**: Users can view detailed information about a specific brewery, including its address, phone number, website, and description.
-   **Add Reviews**: Users can add reviews to breweries, including a rating and description of their experience.

## Technologies Used

-   **React**: The frontend of the application is built using React, a popular JavaScript library for building user interfaces.
-   **Firebase**: Firebase is used as the backend for storing brewery data and user reviews.
-   **Open Brewery DB API**: This API is used to fetch information about breweries, including their details and reviews.

## Getting Started

To run this project locally, follow these steps:

1.  Clone this repository to your local machine.
2.  Install the necessary dependencies using `npm install`.
3.  Obtain a Firebase project and replace the Firebase configuration in the `firebase.js` file with your own project's configuration.
4.  Start the development server using `npm start`.
